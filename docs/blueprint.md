# **App Name**: 領域共備GO

## Core Features:

- 資訊輸入: 讓老師輸入教學領域、會議主題、日期和社群成員等基本資訊。
- 照片展示: 以 2x2 的格式排列顯示上傳的四張照片。
- 照片 AI 描述: 使用 Gemini API 分析照片，根據教學領域和會議主題，為每張照片產生 15-30 字的說明文字。The LLM is used as a tool to augment the information.
- 會議大綱摘要生成: 整合所有資訊和照片描述，產生 300-500 字的會議大綱摘要。
- 報告匯出: 一鍵匯出包含所有內容的教師社群領域會議報告。

## Style Guidelines:

- 主色：使用柔和的淺綠色 (#C8E6C9) 代表教育的成長與希望。
- 輔助色：使用溫暖的米色 (#FFF9C4) 營造親切感。
- 強調色：使用亮橘色 (#FFAB40) 突顯重點按鈕和資訊。
- 採用 RWD 響應式網頁設計，確保在各種裝置上的最佳瀏覽體驗。
- 使用簡潔、現代的圖示，讓介面更易於理解和操作。
- 加入輕微的動畫效果，例如載入指示或滑動效果，提升使用者體驗。
